# Module 1: NETRA - Counterfeit Currency Field Kit

To run this module in standalone CLI mode:
1. Ensure dependencies are installed: `pip install -r modules/netra/requirements.txt`
2. Navigate to the `prahari/` directory: `cd prahari`
3. Run the demo script on a sample image:
   ```bash
   python -m modules.netra.demo --image modules/netra/samples/fake_500_photocopy.jpg
   ```
